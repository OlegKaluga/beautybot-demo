## 📅 Запись на услугу

### Шаг 1: Выбор зала

```
🏛 Выберите зал:

┌─────────────────────┐
│   ✂️ Стрижки       │
├─────────────────────┤
│   💅 Ногти         │
├─────────────────────┤
│   🔙 Назад         │
└─────────────────────┘
```

**О залах:**
- **✂️ Стрижки** — 2 мастера на выбор
- **💅 Ногти** — 1 мастер

### Шаг 2: Выбор мастера (только для стрижек)

```
✂️ Выберите мастера:
🏛 Стрижки

┌─────────────────────┐
│   М.1    │   М.2   │
├─────────────────────┤
│   🔙 Назад         │
└─────────────────────┘
```

*Для ногтей этот шаг пропускается*

### Шаг 3: Выбор услуги

```
💇 Выберите услугу:
🏛 Стрижки, 👤 М.1

┌─────────────────────────────┐
│  Стр.    │  Стр. бороды   │
│  800₽    │  500₽          │
├─────────────────────────────┤
│  Компл.( │   ⬜           │
│  1200₽   │                │
├─────────────────────────────┤
│      🔙 Назад             │
└─────────────────────────────┘
```

**Услуги:**

| Стрижки | Цена | Время |
|---------|------|-------|
| Стрижка | 800₽ | 45 мин |
| Стрижка бороды | 500₽ | 30 мин |
| Комплекс | 1200₽ | 75 мин |

| Ногти | Цена | Время |
|-------|------|-------|
| Маникюр | 1200₽ | 90 мин |
| Педикюр | 1500₽ | 90 мин |
| Покрытие гель-лак | 1800₽ | 120 мин |
| Дизайн ногтей | 500₽ | 30 мин |

### Шаг 4: Выбор даты

```
📅 Выберите дату:
💇 Стрижка (800₽)

┌─────────────────────────────┐
│ Пн │ Вт │ Ср │ Чт │ Пт │ Сб │ Вс │
├────┼────┼────┼────┼────┼────┼────┤
│ 2  │ 3  │ 4  │ 5  │ 6  │ 7  │ 8  │
└────┴────┴────┴────┴────┴────┴────┘
      ⬅️        ➡️       🔙 Назад
```

**Навигация:**
- **⬅️** — предыдущая неделя
- **➡️** — следующая неделя
- **Нажмите на дату** — выбрать

*Доступно 90 дней (13 недель) вперёд*

### Шаг 5: Выбор времени

```
⏰ Время на 2026-03-07:

┌────────────────────────────────────┐
│ ⏰10:00 │ ⏰11:00 │ ⏰12:00 │ ⏰13:00 │
├────────────────────────────────────┤
│ ⏰14:00 │ ⏰15:00 │ ❌16:00 │ ⏰17:00 │
├────────────────────────────────────┤
│ ⏰18:00 │  ⬜     │  ⬜     │  ⬜     │
└────────────────────────────────────┘
      🔙 Назад
```

**Обозначения:**
- ⏰ — свободно (нажмите для записи)
- ❌ — занято
- ⬜ — не работает

### Шаг 6: Ввод данных

**Имя:**
```
✍️ Ваше имя:
```
*Отправьте имя сообщением*

**Телефон:**
```

```
*Отправьте телефон в формате +79991234567*

## 🗓 Мои записи

### Просмотр

1. Нажмите **"🗓 Мои записи"**
2. Вы увидите ближайшую запись:

```
📋 Ваша запись:

🏛 Стрижки
👤 Мастер 1
💇 Стрижка
📅 2026-03-07 10:00

┌─────────────────────┐
│   ❌ Отменить      │
└─────────────────────┘
```

### Отмена записи

1. Нажмите **"❌ Отменить"**
2. Запись будет удалена
3. Слот освободится для других клиентов

## ❓ Помощь

### Быстрые ответы

Нажмите **"❓ Помощь"**:

```
❓ Помощь

• Запись: нажмите «📅 Записаться»
• Отмена: «🗓 Мои записи» → «Отменить»
• Вопрос? Пишите администратору
```

### Частые вопросы

**❓ Как записаться?**

Нажмите **"📅 Записаться"** и следуйте инструкциям бота.

**❓ Как отменить запись?**

Нажмите **"🗓 Мои записи"** → **"❌ Отменить"**.

**❓ Можно ли записаться к конкретному мастеру?**

Да! При записи на стрижки выберите одного из 2 мастеров.

**❓ Как посмотреть цены?**

Нажмите **"💰 Прайсы"**.

**❓ Бот не отвечает, что делать?**

1. Проверьте интернет-соединение
2. Отправьте `/start`
3. Напишите администратору

**❓ Я опаздываю на запись**

Напишите или позвоните администратору (контакты в профиле канала).
